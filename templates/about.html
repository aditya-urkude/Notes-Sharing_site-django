All About Online Notes Sharing

“Online Note Sharing” is Django project used for sharing notes using the website. Notes that the user wants for a particular subject can be found in the website.
This project is written in Python Django and sqlite.
This Website provide student and Teacher login features and Allow Student to Download notes which are uploaded on this site.
To Avoid fake uploades this provide features to admin to verify the notes and grant access to all users.
